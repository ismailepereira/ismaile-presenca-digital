# Ismaile Pereira · Presença Digital

Site institucional / landing page de **Ismaile Pereira** — criação de sites e presença digital para negócios e profissionais de todo o Brasil.

- **Stack:** HTML + CSS + JS puro (arquivo único, sem build). Hospedável em qualquer lugar.
- **Marca:** sol nascente (terracota + verde sobre creme).
- **Objetivo:** gerar contato via WhatsApp.

## ✏️ Como editar (1 lugar só)

No final do `index.html`, dentro do bloco `EDITE AQUI`:

```js
var WHATSAPP  = "5569999999999";   // seu número: 55 + DDD + número
var INSTAGRAM = "...";             // seu Instagram
var YOUTUBE   = "...";             // seu YouTube
```

Trocou o número → todos os botões de WhatsApp passam a funcionar.

## 📁 Estrutura

```
index.html              # o site inteiro

```
